Music Server Status Each server holds its own library; the catalogue is the union of all of them, deduplicated. "Unique in catalogue" counts THAT server's distinct analyzed songs; "Duplicate files" are extra copies of a song already counted (same audio, several files). The "On multiple servers" row is a NEGATIVE adjustment: a song on more than one server is counted in each server row above, so it is subtracted once to avoid double-counting. The "Orphan" row counts songs bound to NO server (a removed server or deleted file leaves the analysis in the catalogue). With both, the "Unique in catalogue" column sums exactly to the total unique songs.
